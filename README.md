# ShareIt

ShareIt – это распределённое приложение для обмена файлами, построенное на Spring Boot 3.3.x и Java 21.  
Проект разделён на два модуля:

- **gateway** – API‑шлюз (REST/GraphQL) с маршрутизацией запросов к сервисам.
- **server** – бизнес‑логика и хранение метаданных файлов.

## Быстрый старт

```bash
# Клонируем репозиторий
git clone https://github.com/azerich4yandex/java-shareit.git
cd java-shareit

# Собираем проект (Maven)
mvn clean install -DskipTests
```

Для запуска в dev‑режиме:

```bash
cd gateway
mvn spring-boot:run -Dspring.profiles.active=dev
```

## Тесты и качество кода

| Инструмент | Что делает |
|------------|-------------|
| **Checkstyle** | Проверка стиля исходного кода согласно `checkstyle.xml`. |
| **SpotBugs**   | Статический анализ на наличие потенциальных ошибок. |
| **JaCoCo**     | Покрытие тестами – минимум 90 % покрытия строк, 60 % ветвей. |

Запустить проверки можно командой:

```bash
mvn verify -Pcheck
```

## CI/CD

В репозитории настроен GitHub Actions pipeline, который:
1. Сборка и запуск юнит‑тестов.
2. Проверка стиля и статический анализ.
3. Генерация отчёта покрытия JaCoCo в формате HTML.

Для локальной работы с пайплайном достаточно запустить `mvn verify`.
